# HG-Lane v2

Code for paper "HG-Lane v2: A Generalizable Generative Paradigm for Controllable Synthesis of Adverse Conditions"(T-PAMI submitted)



## Installation

This part refers to [ComfyUI](https://github.com/Comfy-Org/ComfyUI). For more details or check the errors of installation, you can see in it.

### Clone this repository

Clone this code to your `workspace/HG-Lanev2`. 

Then, clone the ComfyUI and copy it to your workspace.

```bash
cp -r ComfyUI-master/* workspace
```

### Prerequisites

Only test on Ubuntu24.04 with:

- Python >= 3.8 (tested with Python3.12.11)
- PyTorch >= 1.13.1 (tested with Pytorch2.2.0)
- CUDA (tested with cuda12.1)
- Other dependencies described in `requirements.txt`

### Create a conda virtual environment and activate it (conda is optional)

```Shell
conda create -n comfyui python=3.12.11 -y
conda activate comfyui
```

### Install dependencies

```Shell
# Install pytorch firstly, the cudatoolkit version should be same in your system.
conda install pytorch torchvision cudatoolkit=12.1 -c pytorch

# Or you can install via pip
pip install torch==2.2.0 torchvision==0.17.0 torchaudio==2.2.0 --index-url https://download.pytorch.org/whl/cu121

# Then install the requirements.txt
pip install -r requirements.txt
```



## Data preparation

Download CULane datasets. Then extract them to `$CULANEROOT`. Create link to `data` directory. 

```
cd workspace
mkdir -p HG-Lanev2/data
ln -s $CULANEROOT HG-Lanev2/data/culane
```

For our datasets, you should have structure like this:

```
$CULANEROOT/normal               # data folders x6
$CULANEROOT/laneseg_label_w30    # lane segmentation labels
$CULANEROOT/list                 # data lists
```



## Getting Started

### Run the pipeline

```
cd HGLane_ROOT/HG-Lanev2
python culane.py
```

### Seek a better seed

```bash
cd HGLane_ROOT/HG-Lanev2
python process.py
```

Finally, you can get the dataset of our framework. However, you need to generate the `laneseg_label_w30` `list`for the lane detection model to train and test. This can be generated by some tools (eg. seg_label_generate). We also provide our `laneseg_label_w30` `list` to download.
